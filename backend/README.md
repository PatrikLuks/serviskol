# Backend

Node.js + Express/NestJS backend pro aplikaci na správu servisu kol.

## Požadavky
- Node.js
- MongoDB

## Spuštění

1. `npm install`
2. `npm run dev`

## Struktura
- `/models` – MongoDB schémata (User, Bike, ServiceRequest, ...)
- `/routes` – API endpointy
- `/controllers` – logika endpointů
- `/config` – konfigurace (DB, OAuth2, ...)
