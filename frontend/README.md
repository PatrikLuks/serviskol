# Frontend

React frontend pro aplikaci na správu servisu kol.

## Požadavky
- Node.js
- npm

## Spuštění

1. `npm install`
2. `npm start`

## Struktura
- `/src/components` – UI komponenty
- `/src/pages` – jednotlivé stránky (Home, Login, BikeDetail, ...)
- `/src/services` – komunikace s backendem
- `/src/styles` – styly (Tailwind, MUI)
